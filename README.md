# Turkish NLP News & Sentiment Analysis

A transformer-based Natural Language Processing application developed for Turkish news classification and sentiment analysis.

## About the Project

This project performs two main Natural Language Processing tasks:

- Turkish news category classification
- Turkish text sentiment analysis

The models were developed using BERTurk and Hugging Face Transformers. An interactive user interface was created with Gradio to allow users to enter Turkish text and view prediction results with confidence scores.

## Features

- Turkish news classification
- Turkish sentiment analysis
- Transformer-based language models
- Real-time text prediction
- Confidence score visualization
- Interactive Gradio interface

## Technologies

- Python
- BERTurk
- Hugging Face Transformers
- PyTorch
- Scikit-learn
- Pandas
- Gradio
- Google Colab

## Model

The project uses the following Turkish BERT model:

`dbmdz/bert-base-turkish-cased`

## Dataset

The news classification model was trained using a Turkish news dataset containing approximately 42,000 news articles.

## Project File

The complete training, evaluation and interface development process can be viewed in:

`project.ipynb`

## How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Install the required libraries.
3. Run the notebook cells in order.
4. Start the Gradio interface.
5. Enter a Turkish text to view the classification results.
